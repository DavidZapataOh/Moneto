# @moneto/utils

Pure helpers used across the monorepo. No React, no platform deps, tree-shakeable.

## Modules

- **`format.ts`** — currency, date, address formatters (USD, COP, relative time, short address, APY).
- **`clsx.ts`** — class-name composition (Tailwind / NativeWind compat).
- **`utils.ts`** — small generic helpers.
